=======START=======
1. Chạy lệnh npm i để cài các module cần thiết
2. Có sử dụng nodemon và đã config script, có thể chạy server bằng lệnh 'npm start'
3. Server được chạy ở port 3000
4. Truy cập dự án bằng link: http://localhost:3000/
